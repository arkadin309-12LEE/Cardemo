# 💡 2025-09-12_T_미확인 항성 신호

## 핵심 아이디어
- 카드 메모지 형태의 어플리케이션을 만들어볼까?

## 상세 메모
- 카드 형태의 메모를 작성하는 어플리케이션.
	- 카드
		- 새로 만든 카드는 편집 모드가 기본이다.
		- 이미 만들어진 카드는 읽기 모드가 기본이고 편집 모드 선택 시 편집이 가능하다.
		- DECK: 카드 상단 중앙에 위치해 있지만 선택 사항이다.
		- 날짜와 시간: 카드 제일 상당에 들어가는 자동 작성 사항이며 동시에 년/월/일/요일/시간/분 별로 검색이 가능하다.
		  (YYYY-MM-DD(ddd) HH:MM)
		- 제목: 카드의 제목은 선택 사항이다.
		- 내용: 꼭 들어가는 절대 사항이다.
			- 편집 모드시 어플리케이션 상단에 리본 메뉴 형태로 내려온다.
				- 헤더 (1~3)
				- 글자 크기 변경
				- 볼드
				- 이택릭
				- 취소선
				- 밑줄
				- 글 색상
				- 하이라이트
				- 체크 리스트
			- 카드 크기 설정 가능
				- 기본은 어플리케이션의 디스플레이에 맞춰서 자동 맞춤
				- 단 카드의 여백은 1 이하가 될 수 없다.
				- 크기에 맞춰서 작성하는 글이 최대 너비를 초과하면 자동 줄 바꿈 기능
				- 작성하는 글이 설정한 최대 높이를 초과하면 '작성 불가' 팝업 창이 뜨고 새 카드를 만들지에 대한 여부를 정함
					- 이때 카드에 적용했던 이미지 설정은 그대로 따라간다.
			- 방향(가로/세로) 지정 가능
			- 카드 앞/뒤 이미지 설정 가능.
		- 태그: Tag는 작성 가능하며 내용에 들어간다. 이것으로 검색이 가능하다.  
	- 리스트
		- 작성한 DECK과 카드를 불러와 나열해서 볼 수 있는 기능.
		- 리스트의 정렬 순서 기능
			- DECK이 우선 보여주며 카드가 나중에 보여진다.
			- 최근 작성 순서 (기본)
			- 제목 가나다/ABC 순서
			- 내용 가나다/ABC 순서
			- 태그 가나다/ABC 순서
				- 태그가 없는 카드들은 최근 작성 순서로 나열된다.
		- 검색은 작성한 카드들을 검색하는 기능이다.
			- 카드 제목 검색
			- 카드 내용 검색
			- 태그 검색
		- 작성한 카드는 묶어서 DECK에 넣을 수 있다.
			- DECK의 카드들은 '설정'을 통해 DECK과 카드의 이미지를 정할 수 있다.
				- DECK 이미지 설정: 사용자가 선택한 이미지가 DECK 박스에 프린팅 된 이미지가 된다.
				- DECK 명칭 위치 설정: 사용자가 작성한 DECK의 명칭이 DECK의 어디에 위치할지 정한다.
			- 무작위로 카드 1장 뽑기 기능
				- DECK 안에 있는 카드를 섞어서 그 안에 있는 카드 1장을 뽑아 보여주는 기능
				- 단순 재미를 위한 기능이지만 DECK에 묶어서 넣었다면 당연히 이런 기능은 하나 넣고 자신의 뽑기 운을 실험하는 테스트가 있어야 할 것 같다.
